Job Portal Web Application using Flask
A full-stack Job Portal web application built using Python Flask and SQLite.
This project allows users to register, log in, browse jobs, apply for jobs, and track their applications through a dashboard.
Live Demo
https://job-portal-flask-1-zehl.onrender.com/register
Note:
New users should register first. After registration, users are redirected to the login page and then to the dashboard.
Features
User Registration and Login
Session-based Authentication
Add Job Listings
View Available Jobs
Apply for Jobs
View Applied Jobs
Dashboard with Job Statistics
Logout Functionality

How to Run the Project Locally
Clone the repository
git clone https://github.com/KeerthanaReddy-15/job--portal-flask.git
Go to the project folder
cd job--portal-flask
Install dependencies
pip install -r requirements.txt
Run the application
python app.py
Open browser and visit
http://127.0.0.1:5000

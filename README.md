# Job Portal System using Flask

A full-stack Job Portal web application built using Flask and SQLite.

## Features
- User Registration and Login
- Session-based Authentication
- Add Job Listings
- View Available Jobs
- Apply for Jobs
- View Applied Jobs
- Logout functionality

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run the Project
1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000
